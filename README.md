- ✅ **Middleware e RPC**
  - Protocol Buffers (Protobuf)
  - gRPC

- ✅ **Middleware orientati ai messaggi (MOM)**
  - STOMP
  - ActiveMQ

- ✅ **Web API**
  - Microservizi con Flask
